# Writeup

Необходимо написать бота, который способен решить "Сапер 3D".

Для начала рекомендуется изучить стратегии решения Сапера в двухмерном пространстве, чтобы затем применить их на трехмерное пространство.

Подробнее про стратегии решения Сапера можно узнать здесь: [Minesweeper Strategy - Patterns](https://minesweepergame.com/strategy/patterns.php).

Код для бота, который способен решить задание, лежит в файле `solver.py`.
